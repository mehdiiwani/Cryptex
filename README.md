# Cryptex - The Encryption Suite

## Description
Cryptex is a versatile encryption tool that supports a wide range of encryption algorithms including Caesar, Reverse, Atbash, Vigen√®re, Transposition, Rail Fence, and ROT13 ciphers. It provides an intuitive interface for users to encrypt and decrypt text using their chosen algorithm.

## Installation
1. Clone this repository:
    ```
    git clone https://github.com/yourusername/cryptex.git
    ```
2. Navigate to the project directory:
    ```
    cd cryptex
    ```
3. Install the required Python packages:
    ```
    pip install -r requirements.txt
    ```

## Usage
1. Start the Flask server:
    ```
    python app.py
    ```
2. Open your web browser and navigate to `http://localhost:5000`.

3. Select an encryption algorithm from the dropdown menu.

4. Enter the text you want to encrypt or decrypt.

5. Click the "Submit" button to see the result.
